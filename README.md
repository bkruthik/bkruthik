## Featured Projects

### 🎓 [Student Result Management System](https://github.com/bkruthik/Student_Result_Management_System)
CLI application to manage student records and results using Python, MySQL, and OOP.
- Add, update, delete, and view student records and results
- MySQL JOIN queries to fetch student details along with subject results
- Grade logic and pass/fail calculation implemented using OOP

`Python` `MySQL` `OOP` `CLI`

---

### 📚 [Library Book Management System](https://github.com/bkruthik/Library_Book_Management_System)
Full CRUD library management system built with an OOP design.
- Manages book records and member borrowing with complete CRUD operations
- MySQL foreign key relationships linking books, members, and transactions
- OOP classes (Book, Member, Library) with SQL-based search and overdue tracking

`Python` `MySQL` `OOP`

---

### 💰 [Personal Expense Tracker](https://github.com/bkruthik/personal_expense_tracker)
Django web application for tracking daily expenses with full CRUD functionality.
- Users can add, edit, delete, and view daily expenses
- Django models connected to MySQL using the Django ORM
- Forms and views built with Django templates, HTML, and CSS

`Python` `Django` `MySQL` `HTML` `CSS` `Git`
